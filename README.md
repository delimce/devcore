# devCore
Lumen+vue+bulma core to support full stack PHP &amp; Javascript development

## Server requirements:
* PHP >= 7.2
* Mysql/Mariadb >= 5.7
* Composer >= 1.0
* nodejs >=12 & npm >= 6.0

## Components:
### Frontend:
* Vue.js 2
* Lavarel Mix
* Bulma CSS framework

### Backend:
* Laravel/Lumen 6.0 LTS
* PHPUNIT 8.5
* SQLITE 3 (testing)
